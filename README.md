# Routing Protocols (Cisco Packet Tracer)

## Objective
To implement and analyze routing using RIP, OSPF, and EIGRP.

## Topology
3 routers connected via serial links with 3 LAN networks.

## Protocols Implemented
- RIP (Routing Information Protocol)
- OSPF (Open Shortest Path First)
- EIGRP (Enhanced Interior Gateway Routing Protocol)

## Files Included
- rip.pkt → RIP configuration
- ospf.pkt → OSPF configuration
- eigrp.pkt → EIGRP configuration

## Verification
- Successful ping between PC1 and PC3
- Routing tables verified using `show ip route`
